## 🍄 Mis Proyectos Destacados

### [🌿 HONGUERA](https://github.com/Freeak88/honguera)
**El sistema de climatización y control biológico de precisión para cultivo fúngico indoor.**
- **¿Qué es?:** Un controlador climático distribuido basado en **ESP32** y **ESP8266** que gestiona de manera inteligente temperatura, humedad y CO₂.
- **Innovación clave:** Incorpora una máquina de estados segura para el monitoreo redox / ORP del sustrato, previniendo estados de asfixia o anoxia con histéresis calibrada.
- **Stack:** C++ (PlatformIO/Arduino), Mosquitto (MQTT), Node-RED, InfluxDB v2 y Grafana.
- 👉 **[Ver repositorio de Honguera](https://github.com/Freeak88/honguera)**
