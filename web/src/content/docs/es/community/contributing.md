El micelio crece por conexión. No funciona en aislamiento.

## Cómo contribuir

1. Fork → Branch → PR
2. Una idea = un PR. Simple.
3. Convenciones en [CONTRIBUTING.md](https://github.com/Freeak88/honguera/blob/main/CONTRIBUTING.md)

Hardware, firmware, conocimiento de micología, documentación, traducciones — toda contribución suma.

## Licencias

Un proyecto multicapa. Cada capa respira bajo su propia licencia:

| Capa | Licencia | Por qué |
|------|----------|---------|
| 🔩 Hardware (PCB, mecánica) | **CERN-OHL-S 2.0** | Los derivados deben permanecer abiertos |
| ⚡ Firmware | **GPLv3** | Copyleft fuerte. Protege la red |
| 📄 Documentación | **CC BY-SA 4.0** | Atribución + compartir igual |

## Apoyar el proyecto

Honguera está listado en el [directorio FLOSS/fund](https://dir.floss.fund) — un registro curado de proyectos libres/open-source que buscan financiamiento sostenible. Ver [funding.json](https://github.com/Freeak88/honguera/blob/main/funding/funding.json).
